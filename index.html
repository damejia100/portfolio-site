Living ARt
Contributing Developer | GitHub

Mobile app built using React Native and Viro Media. Bringing art to life through AR, VR, animations and sound.

On the frontend, I designed and built the main and list view components. Each screen makes API requests to a PostgreSQL database to load museums and artwork at the selected museum.

Living ARt makes use of Viro Media's image recognition API which made it possible to load AR and VR scenes that included animated versions of paintings. Along with animations, an audio description of the painting is also available to play.

Tech Used: React Native, Viro Media, Node.js, Express.js, PostgreSQL, Sequelize, Heroku, After Effects, AWS
