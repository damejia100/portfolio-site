Living ARt
Contributing Developer | GitHub

Mobile app bringing famous artwork to life through animations, AR, VR, and sound.

Living ARt was built using React Native along with Viro Media and leverages Viro's image recognition API which makes it possible for users to point their camera at artwork found in our database and see an animated version load of top while also playing a recording describing the piece.

On the frontend, I designed and built the main and list view components with each screen making API requests to a PostgreSQL database to load museums and artwork at the selected museum. I also contributed in implementing the image recognition portion of the application.

Built as part of a capstone project with 2 other developers, we all faced the same challenge: never having worked with either AR or VR. Despite this, we rolled out a full stack application within 2 weeks.

Tech Stack: React Native, Viro Media, Node.js, Express.js, PostgreSQL, Sequelize, Heroku, Photoshop, After Effects, AWS

OFFSET
Sole Developer | GitHub

A native mobile app aimed at brining awareness to plastic comsumption by allowing users to track the amount of single use bottles used agains the number of times a resuable bottle is used.

Built using React Native, this cross-platform fullstack application includes screens for signing up, logging in, a main screen to track input consumption,and a personal profile.

This project was started as a 3-day solo hackathon and was inspired by my interest in environmental efforts. I really enjoyed the process of ideation and creation of this application and plan on adding features to it in the future.

Tech Stack: Javascript, React Native, Node.js, Express.js, PostgreSQL & Sequelize
